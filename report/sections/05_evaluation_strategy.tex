\section{Evaluation Strategy}

\subsection{Evaluation Protocol}
To validate the system, we test it on a synthetic dataset derived from high-resolution satellite imagery. UAV camera feeds are artificially generated by applying geometric deformations (scaling, rotation, shearing), noise, and color distortion to sampled map patches. This allows controlled evaluation with known ground-truth pixel coordinates for each generated frame. For each frame, the pipeline estimates UAV position in map pixel coordinates. The estimator output is compared with ground-truth coordinates, while runtime and inlier statistics are collected for performance analysis.

\subsection{Evaluated Metrics}
The primary accuracy metric is Root Mean Square Error (RMSE), which measures the Euclidean distance between estimated coordinates $(\hat{X}, \hat{Y})$ and ground-truth coordinates $(X, Y)$ across $N$ test frames:
\begin{equation}
RMSE = \sqrt{\frac{1}{N} \sum_{i=1}^{N} \left( (X_{i} - \hat{X}_{i})^2 + (Y_{i} - \hat{Y}_{i})^2 \right)}
\end{equation}

In addition to RMSE, we evaluate runtime statistics (mean and standard deviation) and mean inlier ratio to characterize robustness and efficiency of the localization pipeline.
